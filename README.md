# Currency Converter Frontend

This is the frontend application for the Currency Converter project, developed as part of the SoftTeco interview process. It provides a user-friendly interface for converting currencies using real-time exchange rates.

## Features

- Real-time currency conversion
- Supports multiple currencies
- Responsive design
- Integration with backend API for currency rates

## Tech Stack

- React
- TypeScript
- Vite
- Axios (for API requests)

## Setup

### Prerequisites

- Node.js and npm installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/currency_converter_fe.git
   cd currency_converter_fe
   ```
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
   npm run dev
```

4. Open your browser and visit http://localhost:5173 to access the application.

### API Integration

The frontend communicates with the backend API to fetch exchange rates and perform currency conversions. Ensure the backend server is running and accessible.

### Development Context

This project was developed as a technical assessment for SoftTeco, showcasing:

1. Modern React development with TypeScript
2. Efficient state management
3. API integration
4. Responsive UI design

### Development

To build the application for production, run:

```bash
npm run build
```

The output will be in the dist directory, ready to be deployed to a static hosting service.
